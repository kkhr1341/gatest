RENOSYとは

FOLLOW

「住まい探しと資産運用を、もっとカンタンに。」
RENOSYでは、住まいを「借りる」「買う」「売却する」「貸す」、不動産に「投資する」サービスを提供しています。

RENOSYでできること

借りる
東京23区内の高級マンションを厳選して掲載。物件の魅力を最大化するため、3D動画や実際の居住者から集めた口コミなど独自の情報が満載。
READ MORE
買う
都内の選りすぐりの物件を掲載。テクノロジーを活用した物件探しやリノベーション提案まで、お客様の理想の住まい作りをトータルにサポート。
READ MORE
売る
高値・早期売却を実現するため、広告効果の最大化や購入オファーのプッシュ通知機能など、独自の仕組みやAIオンライン価格査定機能を提供。
READ MORE
貸す
高級賃貸物件のWEB集客力を活かし、借主様との早いマッチングを実現。売却・賃貸両方の相談可能。リロケーションなどの短期貸しにも対応。
READ MORE
投資する
厳選した優良物件提案や安くて充実保証付きの賃貸管理、アプリでの物件管理など、不動産×テクノロジーで将来に向けた堅実な資産形成を実現。

実績・認定

  • DX銘柄2020

    経済産業省と東京証券取引所が共同で実施する「デジタルトランスフォーメーション調査2020」において、GAテクノロジーズは「デジタルトランスフォーメーション銘柄」に初選出されました。

  • GOOD DESIGN AWARD 2020

    リアルとテックを融合した最先端の不動産投資サービス「RENOSY ASSET（リノシー アセット）」（現名称：RENOSY 不動産投資）は2020年度グッドデザイン賞を受賞しました。
